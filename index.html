🎛️ Modular Additive Synthesizer

A Max/MSP project exploring additive synthesis, modular sound design, and real-time audio interaction based on principles from Electronic Music and Sound Design.

📌 Overview

This project is a modular synthesizer built in Max/MSP that combines multiple additive synthesis techniques into a flexible and interactive system. Different synthesis modules can be selected and blended in real time using a vector mixing approach, allowing continuous transformation between timbres.

📚 Context & Background

This project was developed based on the book Electronic Music and Sound Design by Alessandro Cipriani and Maurizio Giri. The book introduces digital sound synthesis through both theoretical explanations and practical implementation in Max/MSP.

At the beginning of this project, I had little experience with sound synthesis or Max/MSP. All techniques implemented here were learned progressively and applied through experimentation and iteration.

🧠 Additive Synthesis & Sound Design Principles

Additive synthesis is a fundamental technique in sound design where complex sounds are created by combining simple sine waves, known as partials.

  • Each partial has its own frequency and amplitude
  • The combination defines the overall timbre
  • Parameters can change over time to create dynamic sound

This approach is based on the principle that any periodic waveform can be represented as a sum of sinusoids. It provides precise control over the spectral structure of sound, making it a powerful tool for sound design and experimentation.

🏗️ System Architecture

[4 Synth Modules] → [Selector + Vector Mixer] → [Reverb] → Output

The system consists of four independent synthesis modules. Each module can be assigned a different synthesis method, and their outputs are blended using vector synthesis. The final sound is processed with reverb to add spatial depth.

🔊 Synthesis Modules

Additive Beat Synth

In additive synthesis, beating occurs when two sine waves with slightly different frequencies are combined. The interaction between these frequencies produces a periodic fluctuation in amplitude, perceived as a pulsing effect.

The speed of this pulsation is determined by the difference between the frequencies. For example, combining 440 Hz and 439 Hz produces a 1 Hz beat, resulting in a slow rise and fall in volume. This technique is used to create rhythmic motion and subtle dynamic variation in sound.

Variable Spectrum Synth

Variable spectrum additive synthesis generates complex timbres by combining multiple sine waves, each with independently controlled frequency and amplitude.

Based on the principle that any periodic sound can be represented as a sum of sinusoids, this method allows precise shaping of the spectral content. By dynamically adjusting these parameters over time, it enables the creation of rich, evolving sounds.

Masking Algorithm Synth

The masking method defines a spectral range using two envelope curves that represent the lowest and highest frequencies of the sound.

The space between these boundaries is divided into equal segments, where additional components are distributed. This creates a structured “mask” that controls how partials are arranged, shaping the overall spectrum and influencing how the sound is perceived.

🎚️ Vector Synthesis

Vector synthesis is used to blend the outputs of the four modules in a continuous control space. This allows smooth transitions between different synthesis techniques and enables dynamic exploration of timbre.

🌊 Effects Processing

The final output is processed using reverb from the ABL package, adding spatial depth and enhancing the overall sonic character.
