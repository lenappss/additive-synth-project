🎛️ Modular Additive Synthesizer

A Max/MSP project exploring sound synthesis, modular design, and real-time audio interaction

📌 Overview

This project is a modular additive synthesizer built in Max/MSP. It combines multiple synthesis techniques into a flexible system where different sound engines can be selected, combined, and blended in real time using vector synthesis.

📚 Context & Background

This project is based on the book Electronic Music and Sound Design by Alessandro Cipriani and Maurizio Giri. The book introduces digital sound synthesis using Max/MSP and provides both theoretical and practical approaches to sound design.

Before starting this project, I had little experience with sound synthesis and Max/MSP. All concepts implemented here were learned and applied progressively throughout the project.

🧠 Additive Synthesis (Theory)

Additive synthesis is a fundamental sound synthesis technique where complex sounds are created by summing simple sine waves (partials).

  • Each partial has its own frequency and amplitude
  • The combination defines the timbre
  • It allows precise control over spectral content

This method is powerful but computationally intensive, and it provides deep insight into how sound is constructed.

🏗️ System Architecture

[4 Synth Modules] → [Selector + Vector Mixer] → [Reverb] → Output
  • 4 independent modules
  • Selectable synthesis type per module
  • Vector-based mixing system
  • Final spatial processing with reverb

🔊 Synthesis Modules

Additive Beat Synth

Uses slightly detuned frequencies to create beating effects, producing rhythmic pulsations and interference patterns.

Variable Spectrum Synth

Controls amplitude of partials dynamically, allowing smooth transitions and evolving timbres.

Masking Algorithm Synth

Applies psychoacoustic masking to emphasize or suppress certain frequencies, shaping perception of sound.

🎚️ Vector Synthesis

Vector synthesis blends the outputs of multiple sound sources in a continuous control space. In this project, four modules are mixed dynamically, allowing smooth transitions between different timbres.
