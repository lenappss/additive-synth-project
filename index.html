🎛️ Modular Additive Synthesizer

A Max/MSP project exploring additive synthesis, modular design, and vector mixing

📌 Overview

This project is a modular synthesizer built in Max/MSP using additive synthesis techniques. It combines multiple synthesis modules that can be dynamically selected and blended using a vector-based mixing system.

🧠 Concept

The system is based on additive synthesis, where complex sounds are created by combining sine wave partials. The project expands this concept into a modular, interactive sound design environment.

🏗️ System Architecture

[4 Synth Modules] → [Selector + Vector Mixer] → [Reverb] → Output

🔊 Synthesis Modules

Additive Beat Synth

Uses slightly detuned frequencies to create beating effects and rhythmic pulsations.

Variable Spectrum Synth

Dynamically controls partial amplitudes to enable smooth timbral evolution.

Masking Algorithm Synth

Applies psychoacoustic masking principles to emphasize or suppress frequencies.

🎚️ Vector Synthesis

The system blends four modules using vector synthesis, allowing smooth transitions between different sound sources and creating continuously evolving timbres.

🌊 Effects

Final output is processed using reverb (ABL package), adding spatial depth and atmosphere.
