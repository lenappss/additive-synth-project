🎛️ Modular Additive Synthesizer

A Max/MSP project exploring additive synthesis, modular sound design, and real-time audio interaction.

📌 Overview

This project is a modular synthesizer built in Max/MSP that combines multiple additive synthesis techniques into a flexible system. Different synthesis modules can be selected and blended in real time using vector synthesis.

📚 Context & Background

This project is based on Electronic Music and Sound Design by Alessandro Cipriani and Maurizio Giri.

The book introduces digital sound synthesis through both theory and practical implementation in Max/MSP. All techniques in this project were learned and applied progressively from this resource.

🧠 Additive Synthesis

Additive synthesis creates complex sounds by combining simple sine waves (partials). Each partial contributes to the overall timbre through its frequency and amplitude.

  • Precise control of sound spectrum
  • Flexible and fundamental synthesis method
  • Based on Fourier principles

🏗️ System Architecture

[4 Synth Modules] → [Selector + Vector Mixer] → [Reverb] → Output

🔊 Synthesis Modules

Additive Beat Synth

In additive synthesis, beating occurs when two sine waves with slightly different frequencies are combined. This interaction produces a periodic fluctuation in amplitude, perceived as a pulsing effect.

The speed of the pulsation depends on the frequency difference. For example, 440 Hz and 439 Hz create a 1 Hz beat.

Variable Spectrum Synth

This method builds complex sounds by combining multiple sine waves with independently controlled frequencies and amplitudes.

It allows precise shaping of timbre and enables evolving sound textures over time.

Masking Algorithm Synth

This method defines a spectral range using two envelopes representing the lowest and highest frequencies.

The space between them is divided evenly, creating a structured distribution of partials that shapes perception of sound.

🎚️ Vector Synthesis

Vector synthesis enables dynamic blending of four sound sources using a 2D control space.

By moving a control point, the system smoothly crossfades between sources, allowing evolving and expressive sound transformations.

🌊 Effects

Reverb from the ABL package is applied to add spatial depth and atmosphere.

📖 Work Diary & Development Process

Initial Phase: Learning

At the beginning of the project, I had little to no experience with Max/MSP or sound synthesis. The first stage focused on understanding basic concepts such as signal flow, oscillators, and simple additive synthesis.

Exploration Phase

After gaining basic knowledge, I experimented with combining multiple sine waves and manipulating their frequencies and amplitudes. During this stage, I explored how small changes affect timbre and perception.

Module Development

I then began building individual synthesis modules:

  • Beat synthesis for rhythmic pulsation
  • Variable spectrum synthesis for timbral control
  • Masking algorithm for structured spectral shaping

Each module required testing and iteration to ensure it behaved as expected.

System Integration

The next step was combining all modules into a single system. This involved designing a routing structure and implementing vector synthesis to blend the outputs.

Challenges & Failures

  • Incorrect signal routing leading to no sound output
  • Unstable parameter interactions
  • Difficulty managing multiple modules simultaneously

These issues required debugging, restructuring patches, and simplifying certain parts of the system.

Refinement

In the final stage, I focused on improving usability and stability, testing different parameter ranges, and refining the sound output.

Key Learning Outcomes

  • Understanding of additive synthesis and DSP concepts
  • Experience with modular system design
  • Improved problem-solving and debugging skills

⚖️ Reflection

The project exceeded the initial MVP by evolving into a multi-module system with vector synthesis. The biggest challenge was translating theoretical knowledge into a working system.
